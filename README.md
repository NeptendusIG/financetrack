# financetrack - Finance Tracker

Auteur(s) : Gaétan Lepage

Création : 12/09/2024

Description :

> Opération et réalisation de graphiques sur mes finances

## Prérequis

- **Python** : `Python 3.8+`
- **Dépendences** :
  - `utility`
  - `pandas`
  - `tk`
  - `ttkbootstrap`

## Utilisation

```bash
pip install -r requirements.txt
python -m financetrack
```
